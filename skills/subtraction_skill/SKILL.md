# Subtraction Skill

This skill helps the agent handle requests to subtract two numbers and return their difference.

## Usage

When a user asks to subtract numbers, calculate the difference between them.

## Examples

- "What is 10 minus 3?" → 7
- "Subtract 5 from 20" → 15
- "Calculate 100 - 45" → 55

## Instructions

1. Identify the two numbers in the user's request
2. Determine the order of subtraction (first number minus second number)
3. Return the difference with a clear explanation
