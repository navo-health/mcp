# Simple Addition Skill

This skill helps the agent handle requests to add two integer numbers and return their sum.

## Usage

When a user asks to add numbers, calculate the sum and return the result.

## Examples

- "What is 2 + 3?" → 5
- "Add 10 and 20" → 30
- "Calculate 100 plus 50" → 150

## Instructions

1. Identify the numbers in the user's request
2. Add them together
3. Return the sum with a clear explanation
